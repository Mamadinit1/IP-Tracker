A modern and interactive IP Tracker built with **HTML, CSS, and Vanilla JavaScript**.  
This project automatically detects your public IP address and displays detailed geographical information in a clean and futuristic UI.

---

## 📸 Preview

![IP Tracker Preview](./preview.png)

---

## ✨ Features

- 🌍 Automatically detects your public IP address
- 📍 Displays city information
- 🗺️ Shows country and continent
- 🔗 Shows IP address
- ⚡ One-click tracking
- 📡 Loading state feedback on button

---

## 🚀 How It Works

- On page load or button click, the app fetches IP data
- The response contains location details
- The UI updates instantly with the received information

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript (ES6+)
- Fetch API
